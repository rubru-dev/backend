# Design System: Lumina Digital

## Brand & Style
Lumina Digital is a modern, tech-forward brand that balances professional reliability with a sense of approachable innovation. The brand personality is clear, focused, and energetic. The design style follows a **Corporate / Modern** aesthetic, prioritizing clarity and functional beauty.

## Colors
The color palette is anchored by a sophisticated blue and complemented by vibrant accent tones to guide user attention. 
- **Primary:** #7caee4 (Sky Blue)
- **Secondary:** #be4da0 (Magenta-Plum)
- **Tertiary:** #fdf5ef (Warm Off-white)
- **Mode:** Light

## Typography
The system utilizes **Inter** across all typographic levels to ensure a cohesive, highly legible, and neutral aesthetic. 
- **Headlines:** Bold Inter for strong visual anchors.
- **Body:** 14px and 16px Inter for optimal readability.
- **Labels:** Medium weight Inter for UI metadata.

## Layout & Spacing
The layout is built on a **Fluid Grid** system governed by an 8px base unit (Spacing 2).
- **Desktop:** 12-column grid.
- **Mobile:** 4-column grid with 16px gutters.

## Elevation & Depth
Visual depth is achieved through **Tonal Layers** and soft **Ambient Shadows**. Surfaces are tiered using slight color variations.
- **Base:** Tertiary off-white.
- **Cards:** White surfaces with diffused shadows.

## Shapes
The shape language is **Rounded** (Level 2).
- **Standard (8px):** Buttons and input fields.
- **Large (16px):** Cards and containers.
- **Extra Large (24px):** Modals.

## Components
- **Buttons:** Primary buttons in Sky Blue; Secondary buttons in Magenta-Plum or outline styles.
- **Inputs:** 8px rounded corners with Primary color focus states.
- **Cards:** 16px rounded corners with subtle elevation on Tertiary backgrounds.
- **Chips:** Pill-shaped elements using light tints of the brand palette.
